Blood Cell Classification Using PyTorch And ResNet18
AltayBioAI Project

Description:

This Project Uses A ResNet18 Deep Learning Model To Classify Human Blood Cells Into Four Categories:

- EOSINOPHIL
- LYMPHOCYTE
- MONOCYTE
- NEUTROPHIL

Dataset:

Blood Cells Dataset

Dataset Link:

https://www.kaggle.com/datasets/paultimothymooney/blood-cells

Framework:

- PyTorch
- Torchvision

Model:

- ResNet18

Training Configuration:

- Epochs: 5
- Batch Size: 32
- Image Size: 128 × 128
- Learning Rate: 0.0001
- Optimizer: Adam

Training Results:

- Final Accuracy: 68.68%
- Training Time: Approximately 20 Minutes
- Platform: Google Colab

Included Files:

- Blood-Cell-Classification-GC.ipynb
- Blood-Cell-Classification-JN.ipynb
- Blood-Cell-Classification.py
- Blood-Cell-Classification-Info.txt
- Requirements.txt
- Result Dataset.png
- Result Prediction.png

Model File:

- blood_cell_resnet18.pth

Model Size:

- Approximately 43 MB

Requirements:

- torch
- torchvision
- matplotlib
- numpy
- pillow
- kagglehub

Installation:

pip install -r Requirements.txt

Run:

python Blood-Cell-Classification.py
