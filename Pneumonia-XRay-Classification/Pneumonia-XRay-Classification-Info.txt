Pneumonia X-Ray Classification Using PyTorch And ResNet18
AltayBioAI Project

Description:

This Project Uses A ResNet18 Deep Learning Model To Classify Chest X-Ray Images Into Two Categories:

- NORMAL
- PNEUMONIA

Dataset:

Chest X-Ray Images (Pneumonia)

Dataset Link:

https://www.kaggle.com/datasets/paultimothymooney/chest-xray-pneumonia

Framework:

- PyTorch
- Torchvision

Model:

- ResNet18

Training Configuration:

- Epochs: 5
- Batch Size: 32
- Image Size: 128 × 128
- Learning Rate: 0.0001
- Optimizer: Adam

Training Results:

- Final Accuracy: 77.88%
- Training Time: Approximately 50 Minutes
- Platform: Google Colab

Included Files:

- Pneumonia-XRay-Classification-GC.ipynb
- Pneumonia-XRay-Classification-JN.ipynb
- Pneumonia-XRay-Classification.py
- Pneumonia-XRay-Classification-Info.txt
- Requirements.txt
- Result Dataset.png
- Result Prediction.png

Model File:

- pneumonia_resnet18.pth

Model Size:

- Approximately 43 MB

Requirements:

- torch
- torchvision
- matplotlib
- numpy
- pillow
- kagglehub

Installation:

pip install -r Requirements.txt

Run:

python Pneumonia-XRay-Classification.py
