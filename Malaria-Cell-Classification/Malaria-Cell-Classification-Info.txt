# ALTAYBIOAI PROJECT INFORMATION

Project:
Malaria Cell Classification With PyTorch

Repository:
AltayBioAI

Description:
This Project Demonstrates A Lightweight Convolutional Neural Network (CNN) For Malaria Cell Classification Using Microscopic Blood Cell Images.

The Project Is Designed For Educational, Research, And Biomedical Artificial Intelligence Applications.

MODEL ARCHITECTURE

The Model Uses A Lightweight Convolutional Neural Network.

Layers:

* Conv2D (32 Filters)

* ReLU

* MaxPooling

* Conv2D (64 Filters)

* ReLU

* MaxPooling

* Conv2D (128 Filters)

* ReLU

* MaxPooling

* Fully Connected Layers

* Output Layer
