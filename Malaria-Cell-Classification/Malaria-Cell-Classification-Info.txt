Malaria Cell Classification Using PyTorch
AltayBioAI Project

Description:

This Project Uses A Lightweight Convolutional Neural Network (CNN) To Classify Microscopic Blood Cell Images Into Two Categories:

- Parasitized
- Uninfected

Dataset:

Cell Images For Detecting Malaria

Dataset Link:

https://www.kaggle.com/datasets/iarunava/cell-images-for-detecting-malaria

Framework:

- PyTorch
- Torchvision

Model:

- Lightweight Convolutional Neural Network (CNN)

Model Architecture:

- Conv2D (32 Filters)
- ReLU
- MaxPooling
- Conv2D (64 Filters)
- ReLU
- MaxPooling
- Conv2D (128 Filters)
- ReLU
- MaxPooling
- Fully Connected Layers
- Output Layer

Training Configuration:

- Epochs: 2
- Batch Size: 32
- Image Size: 128 × 128
- Optimizer: Adam

Training Notes:

The Default Number Of Epochs Is Kept Low To Reduce Training Time And Improve Accessibility For CPU And Google Colab Users.

Increasing The Number Of Epochs May Improve Model Accuracy.
